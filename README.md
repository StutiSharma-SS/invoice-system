# Simple Invoice System with GST
This project is a beginner-friendly invoice system built using Python and Flask.

## Features
- Add multiple items
- Calculate subtotal
- GST calculation at 18%
- Display formatted invoice

## How to Run
1. Install Python
2. Install Flask using:
   pip install flask
3. Run the app:
   python app.py

## Custom Feature
GST is automatically calculated and added to the final invoice.
